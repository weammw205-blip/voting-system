# University Voting System

## Description
This project is a simple university voting system that allows users to create accounts, log in, view polls, vote, and see results. It also allows administrators to create new polls. The system ensures that each user can vote only once per poll to maintain fairness.

---

## Features

### 1. User Authentication
- Users can create an account
- Users can log in securely
- User data is stored in the system
- Login validation is applied

---

### 2. Poll Management (User Side)
- Display list of available polls
- Show poll questions
- Open a selected poll for voting

---

### 3. Voting System
- Users can select one option per poll
- Votes are saved in the system
- Duplicate voting is prevented

---

### 4. Results Display
- Count total votes
- Calculate percentages
- Display poll results to users

---

### 5. Admin Features
- Admin can create new polls
- Add poll questions
- Add multiple options for each poll
- Save polls to the system

---

## How It Works
1. User registers and logs in
2. User views available polls
3. User selects a poll and votes
4. System saves vote and prevents duplicate voting
5. Results are calculated and displayed
6. Admin creates and manages polls
